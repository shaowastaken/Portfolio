Sport Scoreboard

Functionality:
The scoreboard displays the scores for a home team and a guest team.
Users can increment the scores by 1, 2, or 3 points for either team.
In case of an error (e.g., accidental button press), users can revert to the previous score.
User Interface:
The site features a clean and intuitive design.
Separate sections for home and guest teams display their respective scores.
Buttons labeled “+1,” “+2,” and “+3” allow users to add points to the teams.
An “Undo” button lets users revert to the previous score.
Implementation:
The site is built using HTML, CSS, and JavaScript.
JavaScript handles score updates and undo functionality.
CSS ensures a visually appealing layout.

JaSh - PixelPainter

JaSh - PixelPainter is a simple Java program for creating pixel art, developed by a beginner.
Don't expect advanced features or a sophisticated interface, but if you're looking for a basic tool to take your first steps into the world of pixel art, JaSh - PixelPainter might be for you.
Main features:
Simple interface: An essential and minimal design, which allows you to get started
right away without getting confused by too many options.
Customizable color palettes: Create, save and manage your own
color palettes to always have the perfect shades for your projects
at hand.
No layer support: You're working on a single layer, so be
careful not to make any mistakes! Fixed Grid: A basic grid to help
you position your pixels with some precision. Export to PNG: You can save your creations in PNG format to share or use elsewhere.
Local Save: Save your work to your hard drive. JaSh - PixelPainter is a first attempt at creating a pixel art program. It doesn't have the advanced features of professional software, but it can be
a fun way to explore pixel art and practice the basics. Perfect
for those who want to experiment without too many pretensions!
